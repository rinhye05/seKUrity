ABOUT THE CLUB

우리는 누구인가
seKUrity

seKUrity는 건국대학교 글로컬캠퍼스 컴퓨터공학과 학생들로 구성된 정보보안 소모임입니다. CTF(Capture the Flag) 대회 참여, 보안 기술 스터디, 웹 해킹, 포렌식, 리버싱 등 다양한 분야의 보안 활동을 함께합니다.

초보자부터 고급자까지 누구나 환영합니다. 같이 배우고, 같이 성장하고, 같이 해킹합시다.

// WHAT WE DO

01
🚩
CTF
국내외 CTF 대회에 팀으로 참여합니다. 각자의 분야에서 익힌 기술을 실전에서 발휘합니다.
02
🌐
WEB HACKING
웹 취약점에 대해 공부하고 분석하며 실습을 진행합니다. 개인 스터디를 통해 각자 역량을 키웁니다.
03
⚙️
REVERSING
바이너리 분석과 리버스 엔지니어링을 개인 스터디로 학습합니다. IDA Pro, Ghidra 등의 툴을 활용합니다.
04
🔐
CRYPTOGRAPHY
암호학 이론과 응용을 개인 스터디로 탐구합니다. 현대 암호 시스템의 원리와 취약점을 분석합니다.
05
🔍
FORENSICS
디지털 포렌식 기법을 개인 스터디로 학습합니다. 메모리 분석, 파일 시스템, 네트워크 패킷 분석 등을 다룹니다.
06
💻
SYSTEM / PWN
시스템 해킹과 익스플로잇을 개인 스터디로 연구합니다. 버퍼 오버플로우, ROP 체인 등 로우레벨 보안을 탐구합니다.
07
🔌
HARDWARE HACKING
임베디드 시스템과 하드웨어 취약점을 개인 스터디로 연구합니다. UART, JTAG, SPI 등 물리적 인터페이스 분석을 탐구합니다.
